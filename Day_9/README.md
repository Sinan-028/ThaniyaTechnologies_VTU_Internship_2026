# Day 9 – Introduction to MongoDB

## Overview

Today I learned the basics of MongoDB, which is a NoSQL database used to store application data in a flexible document format.

## Topics Covered

* What is a Database
* SQL vs NoSQL
* Introduction to MongoDB
* MongoDB structure (Database → Collection → Document)
* Using MongoDB Compass

## Practical Work

* Connected MongoDB Compass to local MongoDB server
* Created a database named **studentDB**
* Created a collection named **students**
* Inserted sample documents into the collection

Example Document:
{
"name": "Sinan",
"course": "MERN Stack",
"skills": ["React", "NodeJS", "Cloud"]
}

## Key Learning

MongoDB stores data in JSON-like documents which makes it very convenient to use with JavaScript and Node.js applications.

## Result

Successfully created a MongoDB database and inserted multiple documents using MongoDB Compass.
