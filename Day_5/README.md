# Day 5 – Student REST API Full Stack

## Backend
Built a Student REST API using Node.js and Express with a modular folder structure (routes, controllers, models).

Endpoints:
- GET /api/students
- POST /api/students
- DELETE /api/students/:id

## Frontend
Created a React application that connects to the backend API using Axios to fetch, add, and delete students.

## Outcome
Developed a mini full-stack application demonstrating frontend-backend communication using REST APIs.
