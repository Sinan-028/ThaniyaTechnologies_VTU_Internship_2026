# Day 1 – User Management System

Today I built a simple full-stack CRUD application using React and Node.js (Express).

## Features
- Add user
- View users
- Delete user

## Tech Used
- React (Frontend)
- Node.js + Express (Backend)
- Axios (API calls)
- Nodemon (auto restart server during development)
- Git & GitHub

## How to Run
Run both servers together:
npm start

Backend:
cd backend  
npm install  
npm run dev  

Frontend:
cd frontend  
npm install  
npm start  

## What I Learned
- How frontend communicates with backend using APIs
- How REST methods (GET, POST, DELETE) work
- Running frontend and backend concurrently
- Using nodemon for development
- Maintaining proper daily folder structure for internship
